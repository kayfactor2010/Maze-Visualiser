# Maze-Generation-and-Pathfinding-Visualiser

Overview:
This project is an interactive maze generation, and pathfinding visualiser built in Python. It helps demonstrate graph algorithms in real-time animation, allowing users to explore how different search strategies operate, and how efficient each is visually.

It supports multiple different maze generation techniques and pathfinding algorithms, making it a useful tool for understand algorithmic behaviour in a dynamic way, while still being visual.

Features:
Maze Generation:
- Prim's Algorithm
- Kruskal's Algorithm
- Recursive Backtracking
Pathfinding Algorithms:
- Depth First Search (DFS)
- Breadth First Search (BFS)
- A* Search Algorithm
- Dijkstra's Algorithm
  
Visualisation Features:
- Real-time algorithm execution animation
- Step-by-step exploration
- Highlights shortest path

Technologies Used:
- Python
- Pygame
- Data Structures
- Algorithm Design
